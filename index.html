Què és una arrow function? A quin ES apareix (Quin any)

Una arrow function es una alternatica compacta a una expresión de función tradicional, pero uno de sus defectos es que es limitado y sno se puede utilizar en todas las situaciónes, esta aparecio en ES5. Diferencias y limitaciones:

  • Función arrow
  • Template Strings
  • Let y Const
  • Generadores
  • Literales octales y binarias
  • Maps y Sets
  • Promesas

Ejemplo de una arrow function:

let ejemploFuncion = () => {
    /* código de la función */
}

Què és un callback? Posa un exemple

Un callback es una función que recibe como argumento otra función y la ejecuta. Ejemplo:

function ejemplo(callback) { 
    callback();
    }

fetch("/ejemplo.txt")
    .then(function(response) {
        /* Código a realizar cuando se cumpla la promesa */
        return response.text();
    })
    .catch(function(error) {
